Exactly XQIB and XQuery

What is the main characteristic of XQuery?

  • XQuery is a W3C standard, just like HTML and XML, and is perfectly tailored for working with HTML, XML and REST. This means more elegant programs and less code.
  • XQuery is a declarative not functional language. XQuery is not as powerful and sophisticated, but can be good for quickly extracting certain elements from a large XML file.
  • Most recent W3C recommendation level 3.1 version for XQuery

What exactly is XQIB? How it can be used with XQuery?

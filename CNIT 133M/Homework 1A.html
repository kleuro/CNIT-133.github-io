Technologies for this class

HTML

HTML, or Hypertext Markup Language, is a markup language for document designed to be displayed in a web browser. HTML is used for a huge variety of things on the web, from building camplex websites that offer email and calendar functions to constructing a simple course website or resume.

CSS

Cascading Style Sheets (CSS) is a simple mechanism for adding style for fonts, colors, spacing to Web documents. By learning CSS you can modify these templates so that they display your colors and styles. You'll have a customized website without a lot of effort.

JavaScript

Designing your web page using JavaScript requires attention to the order in which your code appears and whether you are encapsulating code into functions or objects, all of which impact the order in which the code runs.

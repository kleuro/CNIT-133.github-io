What causes your page to load showly?

  • Unoptimized Images

    • Unoptimized images were the clear winner, impacting 90% of the Alexa Top 1000. Unoptimized images as any image that can be reduced in size without vicual impact to your user, also known as "lossless" optimization.


    Using different image sizes for different devices

  • JPEG - Joint Photographic Experts Group

    • Universal browser and OS support. Fairly low file size. Lossy image compression might lead to poor text readability.

  • PNG - Portable Network Graphics

    • Higher quality(lossless) images and clearly visible text. Learger flie sizes can slow down you website if overused (especially high-resolution images).

  • GIF - Graphics Interchange Format

    • Low file size. Animation support. 8-bit limitation leads to limited image quality
